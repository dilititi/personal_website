# personal_website
This template enables anyone to quickly build their personal website.

# Personal Website · 个人网站

基于 React 19 + Vite 8 构建的双语（EN / 中文）个人作品集网站，包含关于、旅程、作品、书影音、摄影、旅行和联系等模块。

## 前置要求

- [Node.js](https://nodejs.org/) 18+ （建议使用 LTS 版本）
- npm（随 Node.js 自动安装）

## 快速开始

```bash
1. 克隆仓库
git clone https://github.com/dilititi/personal_website.git
cd personal_website

2. 安装依赖
npm install

3. 启动开发服务器
npm run dev
